icms-livechat
=============

Чат для Instant CMS http://instantcms.ru

Зависимости:
Instant CMS 1.10+
Jquery 1.9

Возможности:
* Посыл сообщений 
* Отображение статуса пользователя
* Отображение активности пользователя
* Посыл адрессованных сообщений
* Просмотр истории чата
* Смена цвета пользователя

В разработке:
* Приватные чаты
* Комнаты
